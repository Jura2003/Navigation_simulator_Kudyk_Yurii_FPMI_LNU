
Програмний засіб для моделювання роботи систем внутрішньої навігації (Indoor Positioning Systems) методом трилатерації. Розроблено в рамках магістерської кваліфікаційної роботи.

Автор: Кудик Ю.Р.  



Симулятор демонструє принципи визначення місцезнаходження об'єкта за допомогою трьох джерел сигналу (маяків). Програма дозволяє досліджувати вплив фізичних перешкод (загасання сигналу) та радіошуму на точність навігації.


Для запуску необхідний встановлений Python (версія 3.8 або новіша).

Встановіть бібліотеку Pygame.(pip install pygame)

Керування
Симулятор підтримує інтерактивну взаємодію через мишу та клавіатуру:

Основні дії:
ЛКМ (Ліва кнопка миші): Поставити новий маяк або вибрати існуючий.

ПКМ (Права кнопка миші): Встановити "Цільову точку" (об'єкт, який шукаємо).

ESC: Зняти вибір з маяка.

DEL / Backspace: Видалити вибраний маяк.

C: Очистити цільову точку.

Налаштування параметрів маяка:
Клавіша + / =: Збільшити потужність вибраного маяка (dBm).

Клавіша -: Зменшити потужність.

Параметри середовища (Фізика):
Клавіша N: Збільшити коефіцієнт загасання (n) — імітація стін/перешкод.

Клавіша B: Зменшити коефіцієнт загасання.

Клавіша S: Збільшити рівень шуму (σ) — імітація нестабільного сигналу.

Клавіша A: Зменшити рівень шуму.
